38 /**
39  * \class corenrn_parameters
40  * \brief Parses and contains Command Line parameters for Core Neuron
41  *
42  * This structure contains all the parameters that CoreNeuron fetches
43  * from the Command Line. It uses the CLI11 libraries to parse these parameters
44  * and saves them in an internal public structure. Each parameter can be
45  * accessed or written freely. By default the constructor instantiates a
46  * CLI11 object and initializes it for CoreNeuron use.
47  * This object is freely accessible from any point of the program.
48  * An ostream method is also provided to print out all the parameters that
49  * CLI11 parse.
50  * Please keep in mind that, due to the nature of the subcommands in CLI11,
51  * the command line parameters for subcategories NEED to be come before the relative
52  * parameter. e.g. --mpi --gpu gpu --nwarp
53  * Also single dash long options are not supported anymore (-mpi -> --mpi).
54  */
