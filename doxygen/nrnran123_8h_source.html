32 /* interface to Random123 */
33 /* http://www.thesalmons.org/john/random123/papers/random123sc11.pdf */
34 
35 /*
36 The 4x32 generators utilize a uint32x4 counter and uint32x4 key to transform
37 into an almost cryptographic quality uint32x4 random result.
38 There are many possibilites for balancing the sharing of the internal
39 state instances while reserving a uint32 counter for the stream sequence
40 and reserving other portions of the counter vector for stream identifiers
41 and global index used by all streams.
42 
43 We currently provide a single instance by default in which the policy is
44 to use the 0th counter uint32 as the stream sequence, words 2 and 3 as the
45 stream identifier, and word 0 of the key as the global index. Unused words
46 are constant uint32 0.
47 
48 It is also possible to use Random123 directly without reference to this
49 interface. See Random123-1.02/docs/html/index.html
50 of the full distribution available from
51 http://www.deshawresearch.com/resources_random123.html
52 */
